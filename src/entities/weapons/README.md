# Weapons structure

This directory contains the main weapon class and extracted modules for existing in-game weapons:

- `weapon.ts` — base projectile weapon behavior.
- `fan/` — shotgun fan pattern logic.
- `lightning/` — chain lightning cast and visual effect logic.
- `rotatingBlades/` — rotating blades update and render logic.

Also includes folders for all current weapon archetypes:

- aim
- multi
- fan
- splash
- beam
- four
- lightning
- rotatingBlades
